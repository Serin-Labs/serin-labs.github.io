Flash Firmware

Flash ESPHome or HomeKit firmware to your device directly from the browser using a USB cable. No software installation needed.

1. Select Your Platform

ESPHome integration for Mitsubishi mini splits via CN105. Supports Home Assistant with full climate control, diagnostics, and OTA updates.

2. Select Your Board


Native HomeKit control for Mitsubishi mini splits via CN105.

Supported Boards

  • M5Stack Atom S3 Lite (ESP32-S3)
  • M5Stack NanoC6 (ESP32-C6)

The correct firmware is selected automatically based on your connected device.

3. Flash

Instructions

  1. Connect your device to your computer via USB-C.
  2. Click Install Firmware below and select the serial port.
  3. Wait for flashing to complete, then reset the device.
Your browser does not support Web Serial. Use Chrome or Edge.

This installs a default ESPHome configuration with basic climate control, web server, and OTA support. For a custom configuration with remote sensors, vane control, or diagnostics, use the YAML generator after initial setup.

This installs the native HomeKit firmware. After flashing, the device creates a Wi-Fi access point for setup — no additional software needed.
