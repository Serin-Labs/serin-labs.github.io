Flash Firmware

Flash ESPHome or HomeKit firmware to your device directly from the browser using a USB cable. No software installation needed.

1. Select Your Platform

2. Select Your Board


3. Flash

Connect your device via USB-C, then click the button below.

This installs a default ESPHome configuration with basic climate control, web server, and OTA support. For a custom configuration with remote sensors, vane control, or diagnostics, use the YAML generator after initial setup.

This installs the native HomeKit firmware. After flashing, the device creates a Wi-Fi access point for setup — no additional software needed.

4. Next Steps

Continue to ESPHome Setup

Notes

  • Requires a Chromium-based browser (Chrome, Edge, Brave) with Web Serial support.
  • Use a USB data cable, not a charge-only cable.
  • If your device isn't detected, you may need to install USB drivers.
