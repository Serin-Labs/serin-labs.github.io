How It Compares

There are several ways to add smart control to a Mitsubishi heat pump. This one uses the CN105 — a small wired service port inside most indoor units — for direct, two-way control. Here's how they compare.

Quick Start

  1. Get the Hardware: Board + cable, or a pre-assembled controller. Check compatibility first.
  2. Flash the Firmware: Use the flash tool to install ESPHome or HomeKit-compatible firmware from your browser.
  3. Follow Your Setup Guide: ESPHome setup or HomeKit-compatible setup.
  4. Install in Your HVAC: Power off, connect the CN105-to-Grove cable, tuck the device inside, and restore power.
  5. Multiple Indoor Units? You need one controller per indoor head — each unit has its own CN105 port. Repeat the steps above for each unit, giving each device a unique name.
