Wiring Reference: CN105 to Grove

Pinout and wiring reference for connecting an M5Stack Atom S3, S3 Lite, or NanoC6 to a Mitsubishi mini-split CN105 port.

Wiring Diagram

Connectors

Connection Reference

Wire CN105 Pin Signal Grove Pin Signal
Black Pin 2 GND Pin 1 GND
Red Pin 3 5V Pin 2 5V
White Pin 4 TX Pin 3 RX (GPIO 2)
Yellow Pin 5 RX Pin 4 TX (GPIO 1)

Cable Options

You may use the information above to build your own cable using these parts:

  • JST PAP-05V-S (CN105 side)
  • Grove HY2.0-4P (device side)

Pre-assembled cables coming soon. For a complete shopping list including boards, see the parts list.

Detailed Pinout Reference

CN105 (JST-PAP-05V-S) Pinout

Pin Signal Description
112VNot used
2GNDGround
35V5V Power
4TXData output from HVAC (5V logic)
5RXData input to HVAC (5V logic)

Grove (HY2.0-4P) Pinout for M5Stack Atom S3, S3 Lite, and NanoC6

Pin Signal Description
1GGround (connects to ESP32 GND)
25V5V Power (connects to ESP32 5V/VCC)
3G2RX (connects to ESP32 UART RX pin)
4G1TX (connects to ESP32 UART TX pin)
